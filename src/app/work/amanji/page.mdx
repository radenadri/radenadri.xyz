# Smiling Mind

A focused, minimal design exploration centered around typography, rhythm, and subtle motion.

## Overview

- Role: Frontend Developer
- Stack: Next.js, TypeScript, Tailwind CSS, GSAP
- Timeline: 2024

## Goals

Create a serene visual identity using simple shapes, bold spacing, and gentle transitions to demonstrate layout discipline and component reusability.

## Highlights

- Built with Next.js App Router and React Server Components.
- Typography system tuned for balance and reading comfort.
- Motion choreography using GSAP for subtle, performance-friendly transitions.
